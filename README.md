# 魔方网表日历系统

这是一个基于FullCalendar的日程管理系统，通过魔方网表API实现数据交互，提供了完整的日程管理和展示功能。

## 功能特性

- **日历视图**：直观的日历界面，支持查看、添加、编辑和删除日程
- **用户认证**：支持用户登录验证，提供内外网服务器切换
- **多维度筛选**：按负责人、优先级、状态进行数据筛选
- **多选功能**：支持参与部门和参会人员的多选管理
- **实时同步**：与魔方网表后端数据实时同步
- **响应式设计**：适配不同屏幕尺寸

## 技术栈

- HTML/CSS/JavaScript
- FullCalendar 库
- 魔方网表 API
